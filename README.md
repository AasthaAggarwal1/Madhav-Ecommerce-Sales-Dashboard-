Madhav Ecommerce Sales Dashboard | Power BI

This project is an interactive Power BI dashboard built using the Madhav Ecommerce Sales Dataset to analyze sales performance and business insights.

📊 Dashboard Features
KPI Cards for:
Total Sales Amount
Total Profit
Total Quantity Sold
Average Order Value (AOV)
Quarterly slicer for dynamic filtering
Monthly Profit Analysis
Sales by State
Quantity by Payment Mode
Top Customers by Sales
Quantity by Product Category
Profit by Product Sub-Category
🛠️ Tools Used
Microsoft Power BI
Power Query
DAX
Data Modeling
Interactive Visualizations

📈 Key Insights
Identified monthly profit trends.
Analyzed customer purchasing behavior.
Compared sales performance across different states.
Evaluated preferred payment methods.
Identified the most profitable product sub-categories.
Created an interactive dashboard for business decision-making.
